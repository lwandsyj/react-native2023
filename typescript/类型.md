1. props 类型 PropsWithChildren

    类型定义：

        type PropsWithChildren<P = unknown> = P & { children?: ReactNode | undefined };


    import type {PropsWithChildren} from 'react';

    使用：


        type SectionProps = PropsWithChildren<{
          title: string;
        }>;

        相当于

        type SectionProps = {
            title: string
            children?: ReactNode | undefined  // 可选属性
        }
